# MetStyle - Официальный сайт

Одностраничный сайт с информацией о разработчике, его услугах и проектах.

## Особенности
- Тёмно-фиолетовый переливающийся анимированный фон
- Частицы на фоне с интерактивностью (реагируют на курсор)
- Плавные анимации появления элементов
- Адаптивный дизайн для мобильных устройств
- Два основных экрана: приветствие и информация об услугах

## Используемые технологии
- HTML5
- CSS3 (с анимациями и градиентами)
- JavaScript (для частиц и интерактивности)
- Particles.js для анимированного фона
- Font Awesome для иконок

## Структура проекта
- `index.html` - основная структура сайта
- `style.css` - все стили и анимации
- `script.js` - логика и интерактивность
- `README.md` - этот файл

## Как использовать
1. Клонируйте репозиторий или скачайте файлы
2. Откройте `index.html` в браузере
3. Для публикации на GitHub Pages:
   - Создайте новый репозиторий
   - Загрузите все файлы в репозиторий
   - В настройках репозитория включите GitHub Pages для ветки `main`

## Функциональность
- Главный экран с анимированным текстом приветствия
- Кнопки для навигации:
  - "Прайс" - переход к информации об услугах
  - "Мои работы (отзывы)" - переход в Telegram канал
  - "Написать мне" - переход в Telegram для связи
- Экран с ценами на услуги с категориями:
  - Сайты
  - Софты
  - Обучения/мануалы
- Возможность вернуться на главный экран с экрана цен
- Прокрутка вниз на главном экране автоматически переключает на экран с ценами
